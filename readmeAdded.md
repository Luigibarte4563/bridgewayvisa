## Testimonial Section Update

Added a modern testimonial carousel using Swiper.js to showcase client feedback and success stories.

### Features

* Interactive carousel slider
* Navigation arrows and pagination dots
* Touch and swipe support
* Responsive design for desktop, tablet, and mobile
* Center-focused active card effect
* Smooth transitions and animations

### Dependencies Added

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css" />
<script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js"></script>
```

Swiper.js is used to provide the carousel functionality and interactive user experience within the Testimonial Section.
